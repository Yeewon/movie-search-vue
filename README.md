##  ⚙ 주요 기능

- 영화 검색 API를 통해서, 검색어를 입력해 영화를 검색할 수 있다.
- 검색된 결과를 통해 영화의 상세 정보를 볼 수 있다.
- API 요청(Request)에 대해 로딩 애니메이션을 보여준다. 
- 사용자에게 총 검색 결과 개수를 보여준다.
-  이전페이지, 다음페이지 이동 버튼으로 이동이 가능하다.
