##  ⚙ 주요 기능

- Movie Search API 이용해서 영화를 검색할 수 있다.
- 포스터 🖱️클릭하면 높은 해상도의 포스터와 함께 상세 정보를 볼 수 있다. 
- 이전 페이지, 다음 페이지 이동이 가능하다.
- 로딩 시간의 불만을 줄이기 위해 Skeleton UI 기능을 구현하였다.
